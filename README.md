How to use this free cloud pc! (shout out hamza editz for this cloud pc)

Step 1.

You need to fork this repository so you need an account.

Step 2

Make a new codespace on the forked repo and do sudo su

Step 3

You need to do bash rdp.sh

Step 4

Wait for rdp.sh to do its thing then go to ports then test out the both ports by pressing the internet button (the one that is a circle with lines on it)

Thats it!
